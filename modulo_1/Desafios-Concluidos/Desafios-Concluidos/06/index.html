Tags em HTML

As tags em HTML, são as responsavel por dar uma organização basica dentro de um site, e existem diversas tags para diferentes usos e necessidades. Dentro delas podemos destacar as seguintes.

  • Parágrafo
  • Títulos
  • Imagens
  • Abreviações
  • Código-fonte
  • Listas numeradas
  • Listas com demarcadores
